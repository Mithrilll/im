Лабораторная работа №2. Датчик дискретной случайной величины.

===== Описание работы =====
Реализовать датчик ДСВ в виде функции с параметром p(распределение вероятностей значений ДСВ), возвращающей значение ДСВ, равное i, с вероятностью pi

Датчик реалтзован на основе датчика БСВ, для определения дискретного значения опрделяся интервал которому принадлежит значение из датчика БСВ
по формуле:
r - значение датчика БСВ
sum - сумма всех p от 0 до i-1
если r >= sum && r <= sum + pi, то возвращаем дискретное значение i

Решить заданную игру двух лиц с нулевой суммой в смешанных стратегиях (найти оптимальные смешанные
стратегии и цену игры). Разыграть N игр, вычислить средний выигрыш 1-го игрока, сравнить с ценой игры.

Найдя оптимальные смешанные стратегии и цену игры получил
p = (6/17, 11/17) - распределение вероятнойстей выбора стратегии для первого игрока
q = (8/17, 9/17) - распределение вероятнойстей выбора стратегии для второго игрока
v = 37/17 - цена игры

для определения исхода игры получаем с помощью датчика ДСВ значения i,j(стратегии которые выбрали 1 и 2 игрок) и 
возвращаем выигрыш для первого игрок из матрицы из задания aij

===== Результаты моделирования =====
Разыграв 100000 игр получили средний выйгрыш первого игрока
