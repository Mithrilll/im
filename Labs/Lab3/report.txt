Лабораторная работа №3. Датчик непрерывной случайной величины.

===== Описание работы =====
Реализовать датчик непрерывной случайной величины методом обратной функции, с заданной плотностью вероятностей в виде функции

Датчик реалтзован на основе датчика БСВ, для определения значения использовался метод обратной функции которая потом подается в датчик и также задаются нижняя и верхняя граница для возвращаемого значения:
func - обратная функция заданной функции распределения
a - нижняя граница возвращаемого значения
b - верхняя граница возвращаемого значения

p(x) = (8 * x) / (PI * (4 + x^4)) - заданная плотнойсть распределения
F(x) = 2 * arctg(x^2 / 2) / PI - функция распределения, вычисленная путем интегрирования плотности распределения
func(alpha) = sqrt(2 * tg(PI * alpha / 2)) - обратная функция

Теоритическое мат. ожидание - 1.745372458359863
Теоритическую дисперсию вычислить нельзя так как интеграл расходится, поэтому я решил взять ограничение по x <= 10, в таком случае теоритическая дисперсия - 1.896090124582306
